### How to use app

> Run Flutter
``` terminal
flutter pub get
flutter run 
```

> Generate build_runner
``` terminal
dart run build_runner build
```

> Generate Testing Unit Coverage
``` terminal
flutter test --coverage
```


> Build App Android
``` terminal
flutter buid apk
```
